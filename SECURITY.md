# Política de Seguridad

## Reporte de vulnerabilidades

Si encuentras alguna vulnerabilidad de seguridad en el contenido de este repositorio o en la información presentada en el informe, por favor:

1. **No abras un issue público** para discutir la vulnerabilidad.
2. Envía un correo electrónico a **[tu-email-aqui]** con los detalles.
3. Proporciona toda la información posible para reproducir y entender el problema.

## Proceso de respuesta

1. Confirmaremos la recepción de tu reporte en un plazo de **48 horas**.
2. Investigaremos el problema y determinaremos su impacto.
3. Te mantendremos informado sobre el progreso de la resolución.
4. Una vez solucionado, publicaremos una actualización del informe si es necesario.

## Alcance

Esta política aplica al contenido de este repositorio, incluyendo:

- El informe CTI (`informe_cti.md`)
- Cualquier archivo de documentación relacionado

## Exención de responsabilidad

Este repositorio contiene información de inteligencia de amenazas con fines educativos. No se garantiza la ausencia de errores o información desactualizada. El uso de este contenido es bajo tu propio riesgo.

## Divulgación coordinada

Fomentamos la divulgación coordinada de vulnerabilidades. Trabajaremos contigo para entender el problema y publicar una solución de manera responsable.
